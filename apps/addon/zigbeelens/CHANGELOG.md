# Changelog

## 0.1.0 — 2026-06-14

Initial Home Assistant OS add-on release.

- Read-only Zigbee2MQTT observability console
- Home Assistant Ingress UI
- Multi-network MQTT configuration
- SQLite persistence under `/data`
- Redacted JSON/YAML/Markdown diagnostic reports
- Live dashboard with SSE (polling fallback when SSE is unavailable)
